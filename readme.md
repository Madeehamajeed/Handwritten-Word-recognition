&#x20;**Handwritten Word Recognition using CNN + BiLSTM + CTC**



This project implements an offline Handwritten Word Recognition (HWR) system using a deep learning architecture that combines Convolutional Neural Networks (CNN), Bidirectional Long Short-Term Memory networks (BiLSTM), and Connectionist Temporal Classification (CTC) loss.



The model is trained on the IAM Handwriting Dataset and is capable of recognizing handwritten English words from image inputs.





&#x20;**Project Overview**



The objective of this project is to automatically recognize handwritten words from images without explicit character segmentation.



The system:

\- Extracts visual features using CNN layers

\- Learns sequential dependencies using BiLSTM layers

\- Uses CTC Loss for sequence prediction

\- Supports prediction on custom handwritten images
