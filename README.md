# Handwritten-Word-recognition
Handwritten Word Recognition (HWR) system using CNN and Bidirectional LSTM with CTC Loss

This project implements an end-to-end Handwritten Word Recognition system using a combination of Convolutional Neural Networks (CNN) for feature extraction and Bidirectional Long Short-Term Memory (LSTM) for sequence modeling. It utilizes the CTC (Connectionist Temporal Classification) loss function to align predicted sequences with labels without requiring pre-segmented character images.

Highlights:

Built with TensorFlow and Keras.

Trained on the IAM Words Dataset.

Includes data preprocessing, model training, and a prediction pipeline.
